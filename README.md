# Sistema de Gestão Hospitalar - Padrão Mediator

Projeto desenvolvido para a matéria de Análise e Projeto de Sistemas (APS) do curso de Engenharia de Software no CEUB.

## Objetivo
Demonstrar a aplicação prática do padrão comportamental **Mediator** para desacoplar a comunicação entre Médicos e Pacientes no momento do registro e solicitação de exames clínicos, utilizando a linguagem Java.
